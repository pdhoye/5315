What is Docker

Docker is a tool designed to create and deploy applications by holding them in containers. This allows the developer to package the application and have it run efficiently.

What is Nginx

Nginx is a web server which is used as a reverse proxy, load balancer, mail proxy and HTTP cache.

What is VM (virtual machine)

A virtual machine matches a physical computer by mirroring its architectures and provides functionality for an actual computer.

Who is Cloud Computing Service Provider - Digitalocean

Digitalocean is a cloud infrastructure provider that provides developers with cloud services that helps to deploy and scale applications.
